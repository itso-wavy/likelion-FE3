# Emmet StudyBook

## Basic Syntax
* Child: >
* Sibling: +
* Climb-up: ^
* Grouping: ()
* Multiplication: *
* Item numbering: $
* ID and CLASS attributes
* Custom attributes
* Text: {}
* Implicit tag names
